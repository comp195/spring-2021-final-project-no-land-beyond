# NO LAND BEYOND
### Alex Almanza: a_almanza@u.pacific.edu
### Claryse Adams: c_adams18@u.pacific.edu

## Description
A third-person shooter video game inspired by StarFox 64

## Components
- Windows PC
- Keybord/Mouse OR XBOX Controller
- User Interface
- Unity Engine
- DOTween (external Unity library)
